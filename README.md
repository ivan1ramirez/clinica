# EAC Lucía — Entorno de Aprendizaje Constructivista

**Modelo de Jonassen | Psicología Clínica | EDUCEM 2024**

## 📁 Estructura del Proyecto

```
lucia-eac/
├── index.html          # Página principal (GitHub Pages entry point)
├── css/
│   └── styles.css      # Estilos completos (fuentes, layout, responsive)
├── js/
│   └── script.js       # Dinámica: tabs, modales, foro, gamificación
└── README.md           # Este archivo
```

## 🚀 Despliegue en GitHub Pages

1. Sube todos los archivos a un repositorio de GitHub
2. Ve a **Settings → Pages**
3. En "Source" selecciona la rama `main` y la carpeta raíz `/`
4. El sitio estará disponible en `https://[usuario].github.io/[repositorio]/`

## 🎓 Componentes del EAC (Jonassen, 1999)

| Componente | Implementación en el sitio |
|---|---|
| **El Problema Central** | Caso clínico de Lucía (Sección A) |
| **Casos Relacionados** | Historia clínica cronológica colapsable |
| **Recursos de Información** | Sección C — Repositorio de GPC y guías |
| **Herramientas Cognitivas** | Modales de andamiaje (formulación, IPSRT, psicometría) |
| **Conversación/Colaboración** | Sección D — Foro con gamificación (XP) |
